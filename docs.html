Install CLI

Requirement

  • Window 10, Mac OSX, Linux
  • Node.js, v12 or higher

Install

Mac (Homebrew)

$ brew install krypton-cli

NPM

$ npm install krypton-cli

Yarn

$ yarn install krypton-cli
